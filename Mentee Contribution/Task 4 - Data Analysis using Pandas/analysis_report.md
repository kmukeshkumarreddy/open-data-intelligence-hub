# Analysis Report

## Dataset Overview

The Tips dataset contains restaurant billing information including total bill, tip amount, gender, smoker status, day, time, and party size.

## Data Cleaning

* Checked for missing values
* Removed duplicate records

## Summary Statistics

* Total records after cleaning: 243

* Average tip: 3.00
