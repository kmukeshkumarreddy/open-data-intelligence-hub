# Task 5 - Data Analysis using Polars

## Project Title

**Sales Data Analysis using Polars - Sample Superstore Dataset**

---

## Project Overview

This project demonstrates data analysis using the **Polars** library on the **Sample Superstore** dataset. The objective is to perform data cleaning, preprocessing, exploratory data analysis (EDA), feature engineering, aggregation, visualization, and generate business insights.

---

## Dataset

- **Dataset Name:** Sample Superstore
- **File Format:** CSV
- **Domain:** Retail Sales
- **Library Used:** Polars

The dataset contains information about customer orders, products, sales, discounts, profits, shipping details, and regional sales performance.

---

## Project Objectives

- Load dataset using Polars
- Inspect dataset structure
- Handle missing values
- Remove duplicate records
- Convert date columns
- Perform filtering and sorting
- Feature engineering
- GroupBy and aggregation
- Join operations
- Pivot table creation
- Data visualization
- Export processed datasets
- Generate business insights

---

## Technologies Used

- Python 3.x
- Polars
- PyArrow
- Matplotlib
- Jupyter Notebook

## Key Business Insights

- Technology products contribute significantly to total sales.
- Regional sales performance differs across the business.
- Higher discounts do not always increase profits.
- A few products contribute a major share of total revenue.
- Monthly sales show seasonal trends.
- Customer purchasing behavior varies significantly.
- Furniture products often have lower profit margins.
- Sales trends can support future business forecasting.
