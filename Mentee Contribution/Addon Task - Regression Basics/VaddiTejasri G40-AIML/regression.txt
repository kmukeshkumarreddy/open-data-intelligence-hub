
1.What is regression in machine learning, and when is it used?
Regression is a supervised learning technique used to predict continuous numerical values.
used:predicts the house price


2.What is the difference between simple linear regression and multiple linear regression?
Simple linear regression uses one independent variable, while multiple linear regression uses two or more.


3.What assumptions are made by linear regression?
Linear regression assumes linearity, independence, homoscedasticity, normality of errors, and no multicollinearity.


4.What is the difference between a dependent variable and an independent variable?
The dependent variable is the output being predicted, while the independent variable is the input used for prediction.


5.What do the slope and intercept represent in a linear regression model?
The slope shows the change in the dependent variable for a one-unit change in the independent variable, and the intercept is the predicted value when the independent variable is zero.


6.What is the purpose of the cost function in regression?
The cost function measures prediction error and helps optimize the model parameters.


7.What is Mean Squared Error (MSE), and why is it commonly used?
MSE is the average squared difference between actual and predicted values, commonly used because it heavily penalizes large errors.


8.What is the difference between MAE, MSE, and RMSE?
MAE measures average absolute errors, MSE measures average squared errors, and RMSE is the square root of MSE in the original units.


9.What is the R-squared score, and how should it be interpreted?
R-squared indicates the proportion of variance in the dependent variable explained by the model.


10.What is adjusted R-squared, and why is it useful?
Adjusted R-squared accounts for the number of predictors and helps prevent overestimating model performance.


11.What is overfitting in a regression model?
Overfitting occurs when a model learns the training data too closely, reducing its performance on new data.
and also leads to more noise data


12.What is underfitting in a regression model?
Underfitting occurs when a model is too simple to capture the underlying patterns in the data.


13.What is multicollinearity, and how does it affect regression?
Multicollinearity is high correlation among independent variables, making coefficient estimates unstable.


14.How can multicollinearity be detected?
Multicollinearity can be detected using a correlation matrix or the Variance Inflation Factor (VIF).


15.What is regularization, and why is it used?
Regularization adds a penalty to the model to reduce overfitting and improve generalization.


16.What is the difference between Ridge, Lasso, and Elastic Net regression?
Ridge uses L2 regularization, Lasso uses L1 regularization, and Elastic Net combines both.


17.How does Lasso regression perform feature selection?
Lasso performs feature selection by shrinking some feature coefficients exactly to zero.


18.What is polynomial regression?
Polynomial regression models nonlinear relationships by fitting a polynomial equation to the data.


19.How do outliers affect a regression model?
Outliers can significantly distort the regression line and reduce prediction accuracy.


20.How would you evaluate the performance of a regression model?
A regression model is evaluated using metrics such as MAE, MSE, RMSE, and R-squared.
